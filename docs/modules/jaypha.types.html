Module jaypha.types

Defines some type aliases and traits for convenience

alias string[string] strstr

Using this makes the code easier to read and dramatically reduces the incidence of typos.

alias immutable(ubyte)[] ByteArray

Byte arrays and ranges utilise 'ubyte'. Used for octect streams and binary data. Also a convenient way to avoid UTF conversion issues.

enum isByteRange(R)

Tests if R is a compatible input range of ubyte.

enum utfEnc(C)

Gives a string stating the UTF encoding relevent to the type. The mapping is

CutfEnc
char"UTF-8"
wchar"UTF-16"
dchar"UTF-32"
string"UTF-8"
wstring"UTF-16"
dstring"UTF-32"
