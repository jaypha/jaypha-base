Module jaypha.container.set

This module defines sets.

A set is a group of elements where each element is unique in the set (no duplicates). Sets typically have no ordering.

struct Set(E)

A simple set implementation

This set acts as an ouput range for adding elements. If the element already exists within the set (satisfies '=='), then it is not added.

Properties

size_t size Number of elemetns in the set.
E[] range Elements of the set as a range.

Methods

void put(E e) Adds an element to the set, but only if it is not already an element.
struct IndexableSet(E)

A set implementation where the elements can be accessed via an index.

Properties

size_t size Number of elemetns in the set.
E[] range Elements of the set as a range.

Methods

size_t put(E e) Adds an element to the set, but only if it is not already an element. Returns the index of the element (either newly added, or previously existing)
E opIndex(size_t i) Retrives the element referred to by i
