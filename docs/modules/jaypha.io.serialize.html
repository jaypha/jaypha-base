Module jaypha.io.serialize

Serializes data to and from strings.

It does not support the ability to read/write classes and structs, but does support arrays and associative arrays.

To serialize a value, call serialize!(Type)(value).

Unserializing is a bit more tricky. You need to know the type before hand and unserialise using it, although you unserialize using Variant.

To unserialize a value call unserialize!(Type)(ref const(char)[] str) str will be updated to the remainder of the string and the decoded value is returned.
