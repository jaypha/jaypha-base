Module jaypha.io.print

Provides a set of write like functions inteded to work directly with putput ranges.

T print(Writer,S...)(Writer writer, S args)

Outputs a number of arguments to writer as strings.

Each argument of args is outputted one after the other. Strings and character values will be outputted verbatim. Enumerated type values will have the enumeration label printed. Booleans are outputted as "true" and "false". For all other types, the following is done. If a conversion is possible with to!string(), that function is called. If not, if the argument has a copy(Writer) method, that will be called. Otherwise, if the argument has a toString() method, that will be called. Failing all that, the argument is passed to formattedWrite.

Writer writerOutput range to write to.
S... argsValues to be outputted.
T println(Writer,S...)(Writer writer, S args)

Same as print, but outputs a "\n" at the end.
