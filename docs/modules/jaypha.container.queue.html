Module jaypha.container.queue

This module defines a queue.

A queue is a buffer where the first element added is the first element to be retrieved, or FIFO (First In First Out).

struct Queue(E)

A simple queue implementation

This queue acts as an ouput range for adding elements to the queue, and as an input range for retrieving them.

Properties

bool empty True if the queue has zero elements.
ref E front The first element in the queue
