Module jaypha.container.stack

This module defines a stack.

A stack is a buffer where the last element added is the first element to be retrieved, or LIFO (Last In First Out).

struct Stack(E)

A simple stack implementation

This stack acts as an ouput range for adding elements to the queue, and as an input range for retrieving them.

Properties

bool empty True if the stack is empty
ref E front The top element of the stack

Methods

void put(E e) Adds e to the top of the stack
void popFront() Removes the top element from the stack.
