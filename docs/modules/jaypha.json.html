T[] fromJSON(T:T[])(JSONValue jVal) if(!isSomeString!T)
T[string] fromJSON(T:T[string])(JSONValue jVal)

Extracts a value from a JSONValue object, using templates.

For the scalar and string types, the function will attempt to read the value and then convert it to the desired type.

For arrays and objects (associative arrays indexed by strings), each value in the array will be extracted using the fromJSON function.

Parameters:
JSONValue jValJSONValue object to extract value from.

Returns:
The extracted value, in the type requested.
